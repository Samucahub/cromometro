Criar Projeto:
nest new semstress

cd semstress


Apagar:
src/
├── app.controller.ts ❌
├── app.service.ts ❌

Instalar:
npm install @nestjs/config
npm install class-validator class-transformer


.env:
PORT=3000
DATABASE_URL="postgresql://semstress:semstress@localhost:5432/semstress"
JWT_SECRET=jwt_zemamatudo
JWT_EXPIRES_IN=1d

.env.example:
PORT=
DATABASE_URL=
JWT_SECRET=
JWT_EXPIRES_IN=

Testar:
npm run start:dev

Subir a DB:
docker compose up -d

E Verificar:
docker ps


Instalar Prisma:
npm install prisma --save-dev
npm install @prisma/client

Iniciar:
npx prisma init

E vai criar:
prisma/
├── schema.prisma

Depois de configurar o schema.prisma, criar a primeira migration:
npx prisma migrate dev --name init


Depois editar src/common/prisma/prisma.service.ts e src/common/prisma/prisma.module.ts

Atualizar app.module.ts


Um teste rápido:
npm run start:dev




-Agora o AUTH:

Instalar dependências:
npm install @nestjs/jwt @nestjs/passport passport passport-jwt bcrypt
npm install -D @types/bcrypt @types/passport-jwt


Criar módulos:
nest g module auth
nest g controller auth
nest g service auth

A estrutura:
src/auth/
├── auth.controller.ts
├── auth.service.ts
├── auth.module.ts
├── jwt.strategy.ts
└── dto/
    ├── login.dto.ts
    └── register.dto.ts


Editar os DTOs, editar o AuthService, o AuthController, JWT Strategy e AuthModue

Agora editar o JWT Guard + CurrentUser em common/guards e common/decorators

Atualizar app.module.ts
